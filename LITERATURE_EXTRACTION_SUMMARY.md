## 4. McCall Quality Analysis of SIHAT System

| Attribute | Details |
|-----------|---------|
| **Judul Lengkap** | Analisis Kualitas Sistem Informasi Haji Terpadu Menggunakan Metode McCall |
| **Penulis/Author** | Ahmad Farisi, Rizani Teguh, Ratih Lestari |
| **Institusi** | Program Studi Sistem Informasi, Fakultas Ilmu Komputer dan Rekayasa, Universitas Multi Data Palembang |
| **Tahun Publikasi** | 2022 |
| **Jurnal** | JOINTECS (Journal of Information Technology and Computer Science), Vol. 7 No. 2 |
| **DOI** | Published Online: 31-05-2022 |
| **ISSN** | e-ISSN:2541-6448, p-ISSN:2541-3619 |
| **Metode/Framework** | McCall Software Quality Model; Questionnaire-based survey; Expert weighting; Validity and Reliability testing |
| **Context/Domain** | Integrated Hajj Information System (SIHAT) Quality; Software Quality Measurement; Hajj Service System; PT. Arraudhah Wisata Imani |
| **Hasil Utama (Findings/Contributions)** | • Pengukuran kualitas SIHAT Arraudhah menggunakan metode McCall dengan 30 responden<br>• Quality measurement results:<br>  - Correctness: 58%<br>  - Reliability: 30%<br>  - Efficiency: 19%<br>  - Integrity: 58%<br>  - Usability: 45%<br>• Overall quality value: 41% (range 41-60%), meaning SIHAT Arraudhah is of SUFFICIENT QUALITY<br>• Identified quality dimensions using weighted expert questionnaire with 0.1-0.5 scale |
| **Limitasi yang Disebutkan** | • Pengukuran hanya pada 30 responden (sample size terbatas)<br>• Fokus hanya pada PT. Arraudhah Wisata Imani (satu penyedia haji khusus)<br>• Efficiency score sangat rendah (19%) menunjukkan sistem masih memiliki masalah performa<br>• Reliability score rendah (30%) mengindikasikan ada issue dalam konsistensi sistem<br>• Penelitian tidak memberikan rekomendasi spesifik untuk improvement<br>• Metode McCall mungkin sudah relatif lama untuk standar ISO/IEC terbaru |
| **No. Halaman** | 83-92 |
